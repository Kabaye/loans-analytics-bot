# Обновление 2.0.2

Точечный, но важный релиз по просрочкам и поиску.

## Что поменялось

- **Поиск по ИН в пачке снова стал проще.**
  Если по человеку есть телефон или email, бот показывает их сразу в общем результате без лишних кнопок.

- **FinKit-претензии больше не пересоздаются без необходимости.**
  Если на сегодня уже есть актуальная претензия, бот использует её. В FinKit он идёт за новой только когда сегодняшней претензии ещё нет.

- **Текст FinKit-претензии стал полнее.**
  В документ вернули содержательные куски и из базовой претензии FinKit, и из сопроводительного письма, без лишнего урезания.

- **SMS по просрочке теперь формируется стабильнее.**
  Оставлен сам текст и номер телефона рядом, а рискованный deeplink убран, чтобы генерация не ломалась в Telegram-клиенте.

## Смысл релиза

Меньше лишних действий в поиске, меньше сюрпризов в претензиях и меньше магии там, где нужна просто надежная работа.

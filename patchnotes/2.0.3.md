# Обновление 2.0.3

Ещё один прикладной релиз по поиску и документам.

## Что поменялось

- **Поиск по ИН теперь умеет дотягивать контакты из активных FinKit займов.**
  Если телефон и email не были заранее в кэше, бот пробует аккуратно подтянуть их из актуальных FinKit details и сохранить для следующих поисков.

- **В SMS добавлен deeplink прямо в текст.**
  Кнопку с `sms:` убрали, но сам deeplink теперь показывается строкой, чтобы его можно было скопировать или открыть вручную.

- **Подпись в претензии больше не должна уезжать на совсем пустую страницу.**
  Перед блоком даты / подписи / ФИО документ теперь удерживает с подписью хотя бы хвост последнего текста.

- **`generated-docs` теперь самоочищается.**
  Старые DOCX-файлы старше 5 дней автоматически удаляются при следующей генерации претензии.

## Коротко

Меньше пустого кэша в поиске, меньше мусора в generated-docs и аккуратнее финальный вид претензии.

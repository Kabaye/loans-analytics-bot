# Обновление 2.0.1

Небольшой, но очень прикладной релиз: меньше ручной возни с просрочками, понятнее поиск и аккуратнее претензии.

## Что поменялось

- **Претензии по FinKit теперь обновляются по живым данным.**
  Перед формированием бот заново запрашивает актуальную претензию у FinKit, чтобы суммы, дата и ссылка на документ были свежими именно на сегодня.

- **Шаблон претензии переработан.**
  Текст и структура собраны на основе реальной претензии FinKit и отдельного сопроводительного письма.
  Документ больше не ужимается любой ценой в одну страницу: нормальный шрифт, нормальные поля, несколько страниц — это теперь ок.

- **После генерации FinKit-претензии можно отправить её через FinKit.**
  Если сервис разрешает отправку, бот показывает отдельную кнопку для отправки заемщику.

- **Поиск по ИН получил кнопки `Телефон / email` для каждого результата в пачке.**
  Теперь не нужно гадать, есть ли по человеку доп. контакты: кнопка есть сразу, а если данных нет — бот честно скажет.

- **Источник обогащения данных теперь виден человеку.**
  Если информация приехала из претензии FinKit, архива ЗАЙМись или была введена вручную — бот это прямо подписывает.

- **Убрана лишняя строка `История займов:` в уведомлениях.**
  Блок `Инфа из займов` стал чище и не дублирует сам себя.

## По-человечески

Бот стал меньше «ну вроде что-то сформировал» и больше «вот актуальный документ, вот контакты, вот откуда это взялось».

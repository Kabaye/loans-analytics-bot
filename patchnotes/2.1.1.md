## 2.1.1

- Для `FinKit` в подписках добавлен отдельный фильтр `Доход подтвержден`.
- В уведомлениях `FinKit` теперь показываются дополнительные поля скоринга заемщика:
  - подтвержден ли доход;
  - когда выполнена оценка;
  - какой уровень долговой нагрузки вернул API.
- Эти поля берутся из реального API `FinKit` и теперь отображаются прямо в карточке уведомления рядом с остальной borrower-информацией.

# Обновление 2.0.5

Подкрутили внутреннюю механику refresh, чтобы база увереннее наполнялась актуальными данными.

## Что поменялось

- **Массовое обновление FinKit стало догружать контакты даже для уже известных заемщиков.**
  Раньше часть людей могла остаться без телефона/email, если заемщик уже был известен боту по имени или старому маппингу. Теперь refresh отдельно добирает свежие контакты из live detail и сохраняет их в базу.

- **После полной регенерации выдача по ИН стала полнее.**
  Если в FinKit у займа есть телефон или email, бот теперь заметно чаще показывает их сразу в основном результате поиска.

- **Починили служебный cleanup parser-сессий.**
  Ночной refresh аккуратнее закрывает сетевые сессии и больше не оставляет лишний aiohttp-хвост после большого прогона.

## По-человечески

Это релиз из серии “бот стал менее забывчивым”: если источник знает контакт заемщика, теперь шансы потерять его по дороге стали сильно ниже.
